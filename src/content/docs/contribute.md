## 提交数据

我们非常欢迎开发者和政策专家参与共建。如果您希望补充新的政策或更新现有数据：

1. **GitHub 提交**：
   - 访问我们的 GitHub 仓库。
   - 在 `src/content/policies` 目录下参照现有模板创建新的 Markdown 文件。
   - 提交 Pull Request。

2. **在线反馈**：
   - 在政策详情页下方点击“报告错误”按钮。
   - 填写更正后的内容。

## 加入社区

如果您对 CCUS 政策数字化有更多想法，欢迎联系项目维护者或在 Issue 区发起讨论。
