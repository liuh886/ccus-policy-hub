# GENERATED CONTENT DIRECTORY

DO NOT EDIT files in `policies/` or `facilities/` manually.
These Markdown files are automatically projected from the SQLite Single Source of Truth (`governance/db/ccus_master.sqlite`).

Manual changes will be OVERWRITTEN during the next synchronization.

To update content:
1. Modify the SQLite database.
2. Run `pnpm manage db:export:md`.
