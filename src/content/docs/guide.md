# 快速上手指南

## 1. 政策检索

您可以直接在搜索框中输入关键词，或者通过国家、年份、激励类型等筛选器，快速定位到特定的 CCUS 政策详情页。

## 2. 横向对比

在政策卡片中点击“加入对比”，选定多项政策后进入“对比矩阵”页面。我们将从激励额度、法律责任转移、孔隙使用权等 10 个核心维度为您生成分析表格。

## 3. 地图探索

全球設施交互地图展示了正在运营与计划中的 CCUS 设施。点击地图上的点位，不仅可以查看设施参数，还能看到该项目所适用的关联政策。

## 4. 导出报告

所有的对比矩阵均支持一键生成 PDF 报告或导出 CSV 原始数据，方便您在外部报告中使用。
