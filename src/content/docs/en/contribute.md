# Contribution Guide

We welcome contributions from the community to help keep the CCUS Policy Hub up-to-date and accurate.

## How to Contribute

1. **Submit a New Policy**: Use our GitHub issue template to report a new policy.
2. **Update Data**: Propose changes to existing data through Pull Requests.
3. **Report Errors**: Open an issue if you find any inaccuracies.
