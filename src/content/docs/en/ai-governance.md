# AI-Driven Governance

This project employs advanced AI agents to ensure data integrity and sovereignty in the CCUS Policy Hub.

## Core Principles

- **SQLite Sovereignty**: The database is the single source of truth.
- **Automated Auditing**: Deep audit protocols to verify data consistency.
- **Relational Integrity**: Maintaining dual-directional links between policies and facilities.
