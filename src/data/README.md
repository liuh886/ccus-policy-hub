# GENERATED DATA DIRECTORY

DO NOT EDIT files in this directory manually.
All JSON files here (e.g., `i18n_dictionary.json`) are automatically generated from the SQLite Single Source of Truth (`governance/db/ccus_master.sqlite`).

To update data:
1. Modify the SQLite database (via `scripts/manage.mjs` or SQL tools).
2. Run `pnpm manage db:export:i18n`.
